PROYECTO: Gestion De Granja Avicola Automatizada

Proyecto de Autmatizadas

La finalidad de este proyeto que se apliquen los conocimientos adquiridos de las materias de Gestion de Base de datos y Enbebidos, en las cuales aplicamos almacenamiento y gestion de datos en conjunto con dispositivos enbebidos corespondientemente, con la finalidad de realizar una solucion a un problema planteado el cual debemos aplicar lo visto.

El proyecto a realizar es un sistema automatizado para alimentar aves, en nuestro caso, para realizarlo utilizamos pericos, a los cuales se les realizaria los sistemas automatizados de alimentacion, ventilacio y temperatura los cuales seran monitoreados y controlados desde esta web mediante una key por seguridad. Este sistema mantiene a las aves en un ambien fresco y controlado mediande dispositivos enbebidos y los cuales enviarian informacion hacia una base de datos la cual podriamos consultar desde esta misma pagina. Por lo que podriamos identidicar o categirizar esta web como "Plataforma IoT".
